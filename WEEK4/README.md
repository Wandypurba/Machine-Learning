# 📚 WEEK 4 - Machine Learning (Regression)

Repository ini berisi materi dan praktik pembelajaran **Machine Learning** pada minggu ke-4 dengan fokus utama pada **Regression (Regresi)**.

---

## 📖 Deskripsi

Pada minggu ini, pembelajaran difokuskan pada konsep dasar hingga implementasi **Supervised Learning khususnya Regression**, yang digunakan untuk memprediksi nilai numerik berdasarkan data yang tersedia.

Materi yang dipelajari meliputi:

* Konsep dasar regression
* Implementasi regression menggunakan Python
* Analisis dataset
* Evaluasi model
